Overview

Developed and simulated an active flutter control system for unmanned aircraft wings. Built mathematical models of wing dynamics, designed PID-based controllers with accelerometer feedback and piezoelectric actuators, and validated stability using MATLAB/Simulink analyses. Demonstrated effective flutter suppression at target airspeeds.

What I Built

  • Wing dynamics model suitable for control-oriented simulation.
  • PID-based controller architecture using accelerometer feedback.
  • Simulation workflow in MATLAB/Simulink to evaluate stability across conditions.

Validation

  • Validated stability and controller effectiveness using MATLAB/Simulink analyses.
  • Demonstrated effective flutter suppression at target airspeeds (simulation results).
